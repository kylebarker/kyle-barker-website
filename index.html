My Shelf

React, React-Redux, React-Router, Reactstrap, Discogs API, PostgreSQL, NodeJS, JavaScript

My Shelf is a web application designed to allow users to see their physical music collection on screen. The reason I wanted to create this app is because I collect vinyl records and with some records, the spines are so thin or have little to no text, many of the records are forgotten when browsing for something to listen to. This allows the user to not have this problem as they can use the app to find something to listen to, and then grab it from the shelf and put it on. This application is a full CRUD app that allows you to add, edit, and delete records.

Click here to try adding a record to your shelf. App may take a moment to load.

Project Length: 4 Days

Canvas

NodeJs, Jade/Pug, Socket.io, Bootstrap, jQuery, CSS

Canvas is a backend web application that allows users the ability to draw on a community drawing board. The application is a full CRUD app that allows users to create a profile and chat with other users who are logged in and drawing on the community board. The other note on this project is that Canvas was created in a team of 5 people (including myself). This allowed our team to practice handling merging, merge conflicts, and paired programming.
